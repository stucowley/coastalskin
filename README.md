# Coastal Skin + Beauty

A modern, responsive website for Coastal Skin + Beauty, a boutique skin and beauty clinic located in Goolwa, South Australia.

This project is designed to reflect the brand’s calm, coastal aesthetic while providing a seamless experience for users to explore services and make bookings through the Timely booking system.

## ✨ Features

- Responsive mobile-first design
- Service listings with clear navigation
- Online booking integration through Timely
- Clean, minimal UI aligned with brand identity
- Performance focused
- Store raw HSL values so alpha can be applied dynamically

## 🔨 Tech Stack

- HTML5
- Sass
- JavaScript
- Vite

## 🚀 Getting Started**

### 1. Clone the repository**

```bash
git clone https://github.com/stucowley/coastalskin.git
cd coastalskin
```

### 2. Install dependencies**

```bash
npm install
```

### 3. Run the development server**

```bash
npm run dev
```
